# Linux Shell Script w/Whiptail
Proje bash script dili ile oluşturulmuş olup, aşağıdaki işlemleri gerçekleştirebilmektedir;
<ul>
  <li>Bilgi Girişi</li>
  <li>Kayıt Arama</li>
  <li>Tüm Kayıtları Göster</li>
  <li>Kayıt Sil</li>
  <li>Çıkış</li>
</ul>
